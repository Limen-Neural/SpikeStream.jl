## Task 2: Add Codecov to CI (#17)

### What changes

**`.github/workflows/ci.yml`** — add a new `coverage` job:

```yaml
  coverage:
    name: Code Coverage • Julia 1.11 • ubuntu-22.04
    runs-on: ubuntu-22.04
    steps:
      - uses: actions/checkout@v4

      - name: Setup Julia
        uses: julia-actions/setup-julia@v2
        with:
          version: '1.11'

      - name: Cache Julia artifacts
        uses: actions/cache@v4
        with:
          path: |
            ~/.julia/artifacts
            ~/.julia/compiled
            ~/.julia/packages
            ~/.julia/registries
          key: ${{ runner.os }}-julia-1.11-${{ hashFiles('**/Project.toml') }}
          restore-keys: |
            ${{ runner.os }}-julia-1.11-

      - name: Generate coverage
        run: julia --project=. -e 'using Pkg; Pkg.test(coverage=true)'

      - name: Upload to Codecov
        uses: codecov/codecov-action@v5
        with:
          directory: lcov
          token: ${{ secrets.CODECOV_TOKEN }}
          fail_ci_if_error: false
          flags: unittests
```

**`README.md`** — add coverage badge to badge row:

```markdown
[![codecov](https://codecov.io/gh/Limen-Neural/SpikeStream.jl/branch/main/graph/badge.svg)](https://codecov.io/gh/Limen-Neural/SpikeStream.jl)
```

### Manual steps (after merge)

1. Sign up at https://codecov.io/login/gh → enable `Limen-Neural/SpikeStream.jl`
2. Copy the upload token from Codecov dashboard
3. Add `CODECOV_TOKEN` as a repository secret (Settings → Secrets and variables → Actions → New repository secret)
